# **第一章 · 第四节

加入一个根以后，“维度”是怎么出现的？（终稿）**

上一节中我们看到：
加入一个新数（一个方程的根）会生成一个新的、对四则运算封闭的世界。

这一节要回答的核心问题是：

> **为什么加入一个根会让世界“增加维度”？
> 维度从哪里来？为什么是有限的？**

我们从具体例子入手，逐步归纳一般规律。

---

# **1. 加入一个根后，会产生若干“不可化简”的方向**

在有理数 (\mathbb{Q}) 中加入一个数 (\theta) 之后，
域的封闭性要求我们必须包含所有由 (\theta) 出现的表达式，包括它的各次幂及其线性组合。

于是问题自然变成：

> 这些幂中，到底有多少是“真正独立”的？
> 又有多少可以简化为前面的组合？

独立方向的数量，就是扩张的维度。

---

# **2. (\mathbb{Q}(\sqrt{2})) 为什么是二维？**

观察 (\sqrt{2}) 的幂：

[
(\sqrt{2})^0 = 1,\qquad
(\sqrt{2})^1 = \sqrt{2},\qquad
(\sqrt{2})^2 = 2.
]

关键点在于：

[
(\sqrt{2})^2 = 2 \in \mathbb{Q}.
]

这意味着：

* (1) 是一个方向
* (\sqrt{2}) 是另一个方向
* 二次幂没有产生新的方向，而是回到原来的域里

因此所有更高次幂都能写成：

[
a + b\sqrt{2}.
]

于是 ({1,\sqrt{2}}) 在 (\mathbb{Q}) 上线性独立，并生成整个扩张。

所以：

[
[\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2.
]

这就是“二维”的精确含义。

---

# **3. (\mathbb{Q}(\sqrt[3]{2})) 为什么是三维？**

令：

[
\alpha = \sqrt[3]{2}.
]

观察它的幂：

[
\alpha^0 = 1,\qquad
\alpha^1 = \alpha,\qquad
\alpha^2 = \alpha^2,\qquad
\alpha^3 = 2 \in \mathbb{Q}.
]

这告诉我们：

* (1) 是一个方向
* (\alpha) 是另一个方向
* (\alpha^2) 是第三个方向
* (\alpha^3) 回到旧域，不再产生新的方向

因此所有更高次幂都能化为：

[
a + b\alpha + c\alpha^2.
]

于是 ({1,\alpha,\alpha^2}) 在 (\mathbb{Q}) 上线性独立并生成整个扩张。

因此：

[
[\mathbb{Q}(\alpha):\mathbb{Q}] = 3.
]

加入一个三次方程的根，会自然地形成一个三维世界。

---

# **4. 维度的本质：线性独立方向的数量**

从上面两个例子，我们看到了一般结构：

* 加入一个根后，观察它的幂
* 收集所有无法化简的幂次
* 这些幂次在底域上线性独立
* 它们构成扩张域的一组基底

基底的数量，就是扩张的维度。

这是域扩张的基本思想之一：
**维度反映了新世界中“独立成分”的数量。**

---

# **5. 一般规律：维度 = 最小多项式的次数（定理）**

设 (\theta) 是一个代数数，它满足的最小多项式为：

[
m(x)=x^n + c_{n-1}x^{n-1} + \cdots + c_1x + c_0,
]

那么：

* (\theta^n) 可以被写成 (1,\theta,\dots,\theta^{n-1}) 的线性组合
* ({1,\theta,\dots,\theta^{n-1}}) 在线性上独立

因此：

[
[\mathbb{Q}(\theta):\mathbb{Q}] = n.
]

也就是说：

> **一个根满足几次最小多项式，它的扩张就具有几维。**

前面 (\sqrt{2}) 与 (\sqrt[3]{2}) 的例子正是这个定理的具体体现。

---

# **6. 小结：维度来自“封闭性带来的限制”**

加入一个根后，我们必须让所有涉及它的运算结果都落在域中。
这些运算会产生若干个“无法进一步化简”的方向，它们构成扩张域的基底。

因此：

* 满足二次方程的根 → 二维扩张
* 满足三次方程的根 → 三维扩张
* 满足 (n) 次最小多项式的根 → (n) 维扩张

维度是描述扩张域结构最核心的数字。

下一节，我们将探讨：

> **在这样一个多维世界里，哪些结构可以被保持？
> 当“改变根的位置”却又维持运算结构时，会发生什么？**

这将自然引向伽罗瓦理论的另一条主线：
**域的自同构与对称性的数学本质。**
