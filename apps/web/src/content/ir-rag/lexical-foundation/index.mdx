词搜索到底解决了什么，为什么它这么早出现，却一直没退出舞台，这一组文章集中处理这个问题。

后面很多评分模型都踩在这一层上，这里适合作为进入这条线的入口。

## 这里主要整理的问题

- 倒排索引到底替线性扫描省掉了什么
- 一个最小可用的 Boolean retrieval 系统要有哪些部件
- token / type / term 这几个概念分别落在什么位置
- 文档 → token → term 的规范化流水线怎么组织
- tf-idf + 余弦相似度为什么会成为 ranked retrieval 的早期答案
