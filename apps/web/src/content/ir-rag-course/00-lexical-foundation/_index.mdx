# Series 00 · 词搜索基础

这是 IR 的最早问题，也是后面所有评分模型的底座。

## 学完之后你能

- 解释倒排索引解决了线性扫描的什么问题
- 实现一个最小可用的 Boolean retrieval 系统
- 区分 token / type / term 三个概念
- 写出文档 → token → term 的完整规范化流水线
- 解释 tf-idf + 余弦相似度作为 ranked retrieval 第一套完整答案的位置
