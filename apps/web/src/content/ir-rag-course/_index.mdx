# IR / RAG 主题式系列课程

这是一套**主题式**（topic-oriented）的信息检索与 RAG 学习课程。不按教材章节顺序，而是按一个个完整主题展开——每个 series 跨多本书 + 多篇论文取材，集中讲清一个核心问题。

每篇文章配**页内可运行 Python 代码**：你不需要装环境，点击运行就能看到结果。涉及 GPU 或大模型的部分，外链到 Colab 或自建 JupyterLab。

## 学习路径建议

按顺序：00 → 01 → 04 → 05 → 06 是 RAG 系统的"必经路线"。02 和 03 是支线，可以选读。

按需要：

- 想搞清 BM25 调参的依据 → series-01
- 想做 RAG 工程 → series-00 + series-04 + series-05 + series-06
- 想理解 LTR 在工业搜索的位置 → series-03
- 想理解 dense retrieval 的理论根源 → series-02 + series-04

## 怎么跑代码

文章里的代码块**直接点"运行"按钮**就能跑——这是用 [Pyodide](https://pyodide.org)（Python 在浏览器里）实现的。第一次加载需要约 10 秒下载 Python 环境，之后即点即跑。

涉及 PyTorch / GPU 的章节（series 04-06），文章里会给两个按钮：

- **在 Colab 打开**——免费 GPU，需要 Google 账号
- **在 JupyterLab 打开**——如果你自建了 JupyterLab 服务器
